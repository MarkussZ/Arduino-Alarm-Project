# Arduino Home Intrusion Alarm

An Arduino-based security system designed to simulate a home alarm system using multiple sensors and alert mechanisms.

## Technologies
C++  
Arduino  
Ultrasonic Sensor  
LCD Display  
Keypad Input  

## Features
- Password-protected keypad entry system
- Motion detection using ultrasonic sensors
- Emergency panic button functionality
- Water leak detection monitoring
- Visual alerts via LCD and LED indicators
- Audible alarms using a buzzer
